# 🤖 AI Prompts

Ready-to-use prompts that make AI assistants follow this handbook's standards. Start every session with the [master prompt](master-prompt.md), then add a task-specific one.

| Prompt | Use it to… |
| --- | --- |
| [master-prompt.md](master-prompt.md) | Set the root rules/stack for any AI session |
| [frontend-prompt.md](frontend-prompt.md) | Build any UI/feature to spec |
| [feature-generator.md](feature-generator.md) | Scaffold a full feature module |
| [bug-fix-prompt.md](bug-fix-prompt.md) | Diagnose and fix a bug at the root cause |
| [refactor-prompt.md](refactor-prompt.md) | Improve structure without changing behavior |
| [code-review-prompt.md](code-review-prompt.md) | Review a diff/PR against the standards |
| [docs-generator.md](docs-generator.md) | Write a new handbook chapter |
| [examples-generator.md](examples-generator.md) | Generate a real-world usage example |
| [templates-generator.md](templates-generator.md) | Generate a reusable code template |
| [checklists-generator.md](checklists-generator.md) | Generate a quality checklist |

**How to use:** copy `master-prompt.md` → paste the task prompt → fill in the `<placeholders>` → run.
