# Keyboard Shortcuts

Quick reference for common editor and browser shortcuts (VS Code + Chrome DevTools). `Cmd` = macOS, `Ctrl` = Windows/Linux.

## VS Code — Editing

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + D` | Select next occurrence |
| `Cmd/Ctrl + Shift + L` | Select all occurrences |
| `Alt + ↑ / ↓` | Move line up/down |
| `Shift + Alt + ↑ / ↓` | Copy line up/down |
| `Cmd/Ctrl + /` | Toggle comment |
| `Cmd/Ctrl + Shift + K` | Delete line |
| `Cmd/Ctrl + Enter` | Insert line below |
| `F2` | Rename symbol |
| `Shift + Alt + F` | Format document (Prettier) |

## VS Code — Navigation

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + P` | Quick open file |
| `Cmd/Ctrl + Shift + P` | Command palette |
| `Cmd/Ctrl + Shift + F` | Search across files |
| `Cmd/Ctrl + B` | Toggle sidebar |
| `Cmd/Ctrl + \`` | Toggle terminal |
| `F12` | Go to definition |
| `Alt + F12` | Peek definition |
| `Shift + F12` | Find all references |
| `Cmd/Ctrl + Shift + O` | Go to symbol in file |

## VS Code — Multi-cursor & selection

| Shortcut | Action |
| --- | --- |
| `Alt + Click` | Add cursor |
| `Cmd/Ctrl + Alt + ↑ / ↓` | Add cursor above/below |
| `Cmd/Ctrl + L` | Select current line |

## Chrome DevTools

| Shortcut | Action |
| --- | --- |
| `Cmd/Ctrl + Shift + I` | Toggle DevTools |
| `Cmd/Ctrl + Shift + C` | Inspect element |
| `Cmd/Ctrl + Shift + M` | Toggle device toolbar |
| `Cmd/Ctrl + R` | Reload |
| `Cmd/Ctrl + Shift + R` | Hard reload (no cache) |
| `Esc` | Toggle console drawer |

## Terminal (bash/zsh)

| Shortcut | Action |
| --- | --- |
| `Ctrl + C` | Cancel command |
| `Ctrl + L` | Clear screen |
| `Ctrl + A / E` | Start / end of line |
| `Ctrl + R` | Search history |
| `Ctrl + U / K` | Delete before / after cursor |
